# Air extractor

This project aim at creating an Air extractor setup to avoid 3D print fumes and redirerct it towards an open window.

- Project is based on the following components :

Acryllic tubing option was explored but deemed too costly, so instead I went with a much cheaper transparent plastic sheets.

I used the following reference from amazon : https://www.amazon.fr/dp/B09PNHDQ4R?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1

## Ring connectors

In order the connect the different section of tubing we will use ring connectors there is gonna be 3 types :

- Straight ring connector (wall mounted)
- Straight ring connector with PC Fan for air pulling
- Right angled connector

## Utility gutter

The air extactor is also gonna use a gutter system for :

- Access to utility cables to power on the fans
- Compatible with LED lighting (add-on)
